# Playwright Automation – Product Data Extraction

##  Project Overview
This project automates the extraction of product data from the Iden hiring portal using Playwright (Python).

The script:
- Authenticates the user
- Navigates to the product catalog
- Handles lazy loading via dynamic scrolling
- Extracts product details
- Stores the extracted data in a structured JSON file

---

##  Tech Stack
- Python
- Playwright (Sync API)
- JSON
